Project Overview - This project focuses on analyzing hotel booking demand data to understand customer behavior,booking trends,cancellation patterns and revenue insights.
The project also includes Machine Learning techniques to predict booking cancellations and an interactive PowerBI dashboard for visualization.

Tools and technologies used are-
Python
Pandas
Numpy
Matplotlib
Seaborn
Scikit-learn
Power BI
Jupyter Notebook

Step 1 - Data cleaning - Handled missing values,
Removed duplicate values,
Formatted data types,
Dropped unnecessary columns,
Checked null values 

Step 2 - Feature Engineering - New features were created to improve analysis and modeel performance.
Features Created:
Total Stay = Weekend Nights + Week Nights
Total Guests = Adults + Children + Babies

Step 3 - EDA(Exploratory Data Analysis) 
Performed detailed analysis to identify booking trends and customer behavior

Analysis Performed :
Booking distribution by hotel type
Cancellation analysis
Revenue Analysis
Market segment analysis
Guest type distribution
Country - wise ADR distribution
Correlation heatmap

Visualization Techniques:
Bar charts 
Pie charts
Line charts
Heatmap
Histogram

Step 4 - Predictive Analytics using Logistic Regression:
Machine learning was used to predict booking cancellations.

ML Workflow :
Selected (is_canceled) as a target variable
Performed train-test-split
Trained the model
Evaluated model performance using metrics like Accuracy , AUC Score 

Step 5 - Power BI dashboard : Created an interactive dashboard for hotel booking analysis
KPI cards used are -
Total bookings
Total Revenue
Average ADR
Cancellation Rate

Booking trends visualization
revenue by hotel type
Market segment analysis
Guest distribution charts
Interactive slicers and filters

Conclusion: 
The project successfully analyzed hotel booking trends, customer behavior and revenue performance using Power BI and machine learning. The generated insights 
and prediction model can help hotels reduce cancellations and improve business decisions.
